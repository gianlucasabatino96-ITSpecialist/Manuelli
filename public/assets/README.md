# Assets per Landing Page Alessio Manuelli

## 📸 Immagini da inserire

### Foto di Alessio
Posiziona le foto nella cartella `images/`:

1. **alessio-hero.jpg** - Foto con camicia bianca davanti al Castello di Santa Severa (per Hero Section)
2. **alessio-bio.jpg** - Foto con giacca blu (per sezione "Chi è Alessio")

Dimensioni consigliate:
- Hero: 1920x1080px (landscape)
- Bio: 800x1200px (portrait)

### Loghi delle Liste
Posiziona i loghi nella cartella `logos/`:

1. **logo-noi-con-manuelli.png** - Logo "Noi con Manuelli Sindaco" (verde/turchese)
2. **logo-uniti-per-manuelli.png** - Logo "Uniti per Manuelli Sindaco" (colorato, 3 figure)
3. **logo-orizzonti-comuni.png** - Logo "Orizzonti Comuni" (blu navy, tricolore)

Formato: PNG con sfondo trasparente
Dimensioni: minimo 300x300px

## 🎨 Palette Colori Ufficiale

- Verde: `#2fa56a`
- Ciano: `#308d95`
- Azzurro intenso: `#0c6393`
- Azzurro chiaro: `#34b6d7`

## ✅ Quando avrai i file

1. Salva le foto in `assets/images/`
2. Salva i loghi in `assets/logos/`
3. Il wireframe HTML è già configurato per caricarli automaticamente
